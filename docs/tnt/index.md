# The TnT engine blog.

A WIP(work in progress) 2d game engine written in C++20 by Terens Tare which uses Lua, SDL2 and JSON.

Planned features:
* Scripting support.
* Hot code reloading.
* Live code editor.
* Fast logging.
* Data serialization.
* Tiled map support.
* Fast Widgets library.
